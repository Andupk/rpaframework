JSON¶

class RPA.JSON.JSON¶

Bases: object

JSON is a library for manipulating JSON objects. Locating specific elements in the structure is done using JSONPath.

ROBOT_LIBRARY_SCOPE = 'GLOBAL'¶

  • expr – jsonpath expression
