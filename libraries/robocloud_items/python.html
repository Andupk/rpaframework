class RPA.Robocloud.Items.Items(load_env=True, default_adapter=<class 'RPA.Robocloud.Items.RobocloudAdapter'>)¶

Bases: object

A library for interacting with RPA work items.

WorkItems is a collection of keywords for handling data that is moved between different processes and Robot Framework tasks. It allows storing and restoring values to/from cloud or file based storage, and manipulating their contents.

Parameters

  • load_env – Automatically load work item using environment variables

  • default_adapter – Set default adapter if not overriden by environment

Environment variables:

  • RPA_WORKITEMS_ADAPTER: Import path to adapter, e.g. “mymodule.MyAdapter”

  • RC_WORKSPACE_ID: Current default workspace ID

  • RC_WORKITEM_ID: Current default work item ID
