open_available_browser(url: str, use_profile: bool = False, headless: bool = False, maximized: bool = False, browser_selection: Any = 'AUTO') → int¶

Open available browser

Keywords opens the first available browser it can find from the system in preferred order or given browser (browser_selection)

Steps:

  1. Get order of browsers

  2. Loop list of preferred of browsers

    1. Set webdriver options for browser

    2. Create webdriver using existing installaiton

    3. (If step b. failed) Download and install webdriver, try again

    4. (If step c. failed) Try starting webdriver in headless mode

  3. Open url

If unable to open browser raises BrowserNotFoundError.

Details on Safari webdriver setup.

Parameters

  • url – address to open

  • use_profile – set browser profile, defaults to False

  • headless – run in headless mode, defaults to False

  • maximized – run window maximized, defaults to False

  • browser_selection – browser name, defaults to AUTOMATIC_BROWSER_SELECTION

Returns

index of the webdriver session
