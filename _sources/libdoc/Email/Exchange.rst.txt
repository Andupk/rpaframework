###################
Robot Framework API
###################

***********
Description
***********

:Library scope: 

Library for interfacing with Microsoft Exchange Web Services (EWS).


********
Keywords
********

:Authorize:
  :Arguments: username: str, password: str, autodiscover: bool=True

  Connect to Exchange account


:List Messages:
  :Arguments: count: int=100

  List messages in the account inbox. Order by descending
  received time.


:Send Message:
  :Arguments: recipients: str=None, subject: str=, body: str=, attachments: str=None, html: bool=False, images: str=None, cc: str=None, bcc: str=None, save: bool=False

  Keyword for sending message through connected Exchange account.

  Email addresses can be prefixed with `ex:` to indicate Exchange
  account address.

  Recipients is ``required`` parameter.
