###################
Robot Framework API
###################

***********
Description
***********

:Library scope: 

Library for manipulating Word application.

********
Keywords
********

:Close Document:
  :Arguments: save_changes: bool = False

  Close the active document (if open).

:Create New Document:
  Create new document for Word application

:Export To Pdf:
  :Arguments: filename: str

  Export active document into PDF file.


:Get All Texts:
  Get all texts from active document


:Open Application:
  :Arguments: visible: bool = False, display_alerts: bool = False

  Open the Word application.


:Open File:
  :Arguments: filename: str = None

  Open Word document with filename.


:Quit Application:
  :Arguments: save_changes: bool = False

  Quit the application.

:Replace Text:
  :Arguments: find: str, replace: str

  Replace text in active document


:Save Document:
  Save active document

:Save Document As:
  :Arguments: filename: str, fileformat: str = None

  Save document with filename and optionally with given fileformat


:Set Footer:
  :Arguments: text: str

  Set footer for the active document


:Set Header:
  :Arguments: text: str

  Set header for the active document


:Write Text:
  :Arguments: text: str, newline: bool = True

  Writes given text at the end of the document
